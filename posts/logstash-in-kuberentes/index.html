Logstash In Kubernetes#

When perusing the official Logstash documentation at https://www.elastic.co/guide/en/logstash/current/running-logstash-command-line.html#command-line-flags ,we found that there are a lot of configuration files.Deploying a logstash in a Kubernetes environment could be a challenge at first glance, especially when you’re not familar with it.

Actually, you might as well keep focus on what configurations Logstash loads if you’re planning to run an MVP (Minimum Viable Product) with Logstash.

Mount A Configuration File#

Mounting a configuration via a configMap into a container enables Logstash to load it with creating the container.
